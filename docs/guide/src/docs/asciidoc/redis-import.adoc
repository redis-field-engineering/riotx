[[_redis_import]]
= Redis Import

The `redis-import` command reads data from a Redis database and writes it to another Redis database.

The basic usage is:

[source,console]
----
riotx redis-import [OPTIONS] [REDIS COMMAND...]
----

For the full usage, run:
[source,console]
----
riotx redis-import --help
----
